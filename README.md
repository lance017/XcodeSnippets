## Install

Go to the project directory and run:

```shell
python build_snippets.py
```

## Usage

For objective-c:

Type the tip word and tap `esc` in `.h` or `.m` files.

![oc](./1.gif)

For swift:

Type `swift` and tap `esc` in `swift` files.
